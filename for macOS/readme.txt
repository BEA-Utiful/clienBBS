clienBBS for OS X 

# 설치

clienBBS_macOS.zip 압축을 해제 하여 주세요

/clienBBS 에 압축을 풀었을 경우
/clienBBS/clien
/clienBBS/clienBBS.terminal 두 파일이 생깁니다

# 실행

Finder에서 ClienBBS.terminal 을 클릭해주세요
프롬프트에서 /clienBBS/clien (엔터)

혹은 실행파일을 바로 더블 클릭하시어 실행하시고
원하는 터미널 디자인으로 사용하셔도 됩니다

좋은 테마 세팅이 있으시면 공유해주세요!!!
